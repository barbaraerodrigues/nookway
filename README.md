O projeto consiste no desenvolvimento de um aplicativo de turismo social com foco em experiências locais personalizadas. Ele permite o cadastro e login de usuários, visualização de tours, avaliação de experiências, edição de perfil e envio de mensagens de contato. O sistema foi desenvolvido para fins acadêmicos como parte de um Trabalho de Conclusão de Curso.

Funcionalidades principais: cadastro e autenticação de usuários, listagem e detalhamento de tours, upload de imagens, sistema de avaliação, edição de perfil, página de contato e integração com banco de dados MySQL.

Tecnologias utilizadas: PHP 8+, MySQL/MariaDB, HTML5, CSS3, JavaScript e XAMPP como ambiente de execução local.

Como rodar o projeto localmente: Instale o XAMPP a partir do site oficial e coloque a pasta do projeto dentro de C:\xampp\htdocs\tcc_turismo. Abra o XAMPP Control Panel e inicie Apache e MySQL. No navegador, acesse http://localhost/phpmyadmin , crie um banco de dados com o nome tcc_turismo e importe o arquivo SQL localizado na pasta do projeto. Depois disso, acesse o sistema pelo endereço http://localhost/tcc_turismo

Estrutura geral de diretórios: templates, uploads (com subpastas para usuários e tours), css, public e arquivos PHP principais como cadastrar.php, atualizar_perfil.php, avaliar.php, contato.php e index.php.
